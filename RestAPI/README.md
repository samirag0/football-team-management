# Football Team Management System

## Overview

A football management platform developed using Node.js, Express, MySQL and JavaScript.

Features:

- Player management
- Team management
- Match fixtures
- Dashboard statistics
- League tables
- News integration
- User authentication
- Role-based access
- Email notifications

## Technologies

Backend:
- Node.js
- Express
- MySQL

Frontend:
- HTML
- CSS
- JavaScript

## Installation

1. Import RestAPI-SQL.sql into MySQL

2. Backend

```bash
cd RestAPI
npm install
node app.js
```

3. Frontend

Open WebClientApplication/index.html
