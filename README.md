# Football Team Management System

## Overview

A football management platform developed using Node.js, Express, MySQL and JavaScript.

## Cloud 
Frontend URL:
https://6a0d0ff391e89e87e59952a4--euphonious-belekoy-6bfe3e.netlify.app/

Backend API:
https://football-team-management-kojs.onrender.com

Deployment stack:
Frontend: Netlify
Backend: Render
Database: Aiven MySQL

Features:

- Player management
- Team management
- Match fixtures
- Dashboard statistics
- League tables
- News integration
- User authentication
- Role-based access
- Email notifications

## Technologies

Backend:
- Node.js
- Express
- MySQL

Frontend:
- HTML
- CSS
- JavaScript

## Installation

1. Import RestAPI-SQL.sql into MySQL

2. Backend

```bash
cd RestAPI
npm install
node app.js
```

3. Frontend

Open WebClientApplication/index.html
